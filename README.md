# Mackeroni Skills

A collection of specialized agent skills for Claude Code and Gemini CLI.

## Included Skills

- **[ux-designer](./ux-designer)**: @thedotmack's personal landing page design system for building high-conversion marketing and product sites.
- **[claude-code-plugin-release](./claude-code-plugin-release)**: Automated semantic versioning and release workflow specifically for Claude Code plugins.
- **[release-and-auto-changelog](./release-and-auto-changelog)**: A generic, automated release workflow for any repository, including automated GitHub releases and changelog generation.

## Installation

To use these skills, clone this repository into your skills directory:
```bash
cp -r <skill-directory> ~/.claude/skills/
```
